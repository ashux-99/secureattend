# Changelog

All notable changes to SecureAttend will be documented in this file.

## [1.0.0] - 2025-01-27

### Added
- PKI-based QR code generation and verification
- Student attendance portal with 10+ subjects
- QR code decoder with cryptographic validation
- Minimal white UI built with PySide6
- Docker support with Dockerfile and docker-compose
- CI/CD workflows with GitHub Actions
- Unit tests for crypto and validation functions
- File-based attendance record storage
- Configuration management system
- Comprehensive documentation

### Security
- RSA 2048-bit key encryption
- PKI signature verification for QR codes
- Secure key management system
